Follow these simple steps to make this page live so you can send the link to Venuri!

### 1. Create a GitHub Account
If you don't have one, sign up at [github.com](https://github.com/).

### 2. Create a New Repository
1. Click the **+** icon in the top right and select **New repository**.
2. Name it something like `for-venuri` or `my-sudu`.
3. Keep it **Public**.
4. Click **Create repository**.

### 3. Upload Your Files
1. On your new repository page, click the **uploading an existing file** link.
2. Drag and drop these files from your computer:
   - `index.html`
   - `style.css`
   - `script.js`
   - `2025_05_15_11_17_IMG_2013.jpg` (Make sure the photo is uploaded too!)
3. Click **Commit changes**.

### 4. Enable GitHub Pages
1. Go to the **Settings** tab of your repository.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment > Branch**, select `main` (or `master`) and click **Save**.
4. Wait about 1-2 minutes.
5. Refresh the page, and you will see a link like: `https://yourusername.github.io/repository-name/`

### 5. Send the Link!
Copy that link and send it to her. She can open it on any phone or laptop!
